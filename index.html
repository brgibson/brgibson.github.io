About Ben

Ben is a software engineer at Orbitz Worldwide. He lives with his wife in the Chicago area.

Why did Ben create this site?
The main reason Ben created this site is to have a place to experiment with code outside of work. He also has a number of old projects which he would like to clean up and make public.
How pleased is Ben with this website?
None of the experiments on this site are what he would consider to be his best work, but he believes it is better to push something. Everything can be enhanced later. It's far better to push what you have completed, instead of waiting for perfection, which is probably impossible to acheive.
